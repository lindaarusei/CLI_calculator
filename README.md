# CLI_calculator
A simple command-line calculator implementing basic arithmetic operations with error handling.
 Python Command-Line Calculator

## Features

- **Basic Operations**: 
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Exponentiation (^)
  
- **Error Handling**:
  - Invalid number inputs
  - Division by zero
  - Invalid operations
